# DPP Construccion — Ruta de Implementacion

Guia interactiva para fabricantes de productos de construccion que necesitan implementar el **Pasaporte Digital de Producto (DPP)** conforme al Reglamento europeo CPR 2024/3110 y al ESPR 2024/1781.

## Contenido

- **5 fases** de implementacion con pasos detallados y enlaces a legislacion oficial (EUR-Lex, BOE, ECHA)
- **10 fichas de familias** de productos (hormigon, acero, ventanas, cemento, aislamiento, ceramica, madera, plasticos, prefabricados, suelos)
- **FAQ** con las preguntas mas frecuentes de fabricantes
- **+60 enlaces** a fuentes oficiales y guias tecnicas
